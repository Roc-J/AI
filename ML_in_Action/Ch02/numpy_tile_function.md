# numpy函数：tile函数用法 #
tile函数位于python模块 numpy.lib.shape_base中  
功能是重复某个数组。比如tile(A,n)，功能是将数组A重复n次，构成一个新的数组，我们还是使用具体的例子来说明问题：  

1.引入numpy下的所有方法

	from numpy import *

2.我们创建一个a,使用tile来创建b,注意看b的数据结构  

	a = [0,1,2]  
	b = tile(a,2)  
	b  
	array([0, 1, 2, 0, 1, 2])

3.假如我们输入一个元组（1，2），我们会得到一样的结果，与上面相同的b  

	>>> b = tile(a,(1,2))  
	>>> b  
	array([[0, 1, 2, 0, 1, 2]])

4.当然，我们想要a变成一个二维数组，就要换一种重复的方式了
 
	>>> b = tile(a,(2,1))  
	>>> b   
	array([[0, 1, 2],
      	 [0, 1, 2]])
